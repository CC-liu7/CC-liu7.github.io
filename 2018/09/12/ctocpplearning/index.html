c to c++

运算符重载

对新数据类型定义运算,将他们理解成函数

返回类型　operator 运算符符号　（参数列表）

对于重载的运算符成员函数与非成员函数有所不同

类

  • 初始化函数　

  • 返回自引用的成员函数

  • 成员函数重载运算符

  • 构造函数

    在定义类时会自动调用构造函数

    如果未定义构造函数，编译器会自动生成默认构造函数

    在一个类中可定义多个不同的构造函数（函数重载）

    或给参数以默认值

  • 析构函数

    后定义的变量会先销毁

    不要给析构函数提供参数　virtual ~类型名

  • 访问控制与接口　

    使用class定义的类默认其成员为private,外部函数无法访问

    public 为一个类的接口

  • 拷贝构造函数和赋值运算符

    拷贝构造函数：定义一个类对象时同时用同类型的另外对象初始化

    赋值运算符：一个对象赋值给另外一个对象

    未定义时，编译器会自动生成

    但使用默认拷贝构造函数或赋值运算符时可能会出错（内存分配问题）

  • 类体外定义方法

    要在类体中声明

    在类外实现时要在函数前加上类名作用域

  • 类模板

  • string类

  • vector类模板

  • 继承

    派生类从１个或多个父类/基类中继承,继承父类的属性和行为，也有自己的属性和行为。

    继承方式：private,public..

  • 虚态与多态

    使用基类指针（多态性）

    在基类函数前加上virtual关键字将其变成虚函数,基类指针会随其实际指向类型调用其函数

  • 多重继承

    从多个类派生出一个类

  • 纯虚函数，抽象类

    纯虚函数：函数体＝０

    包含有至少一个纯虚函数的类为抽象类

    不能定义抽象类的对象。

    纯虚函数派生出的类必须实现纯虚函数才可以定义派生类的对象
